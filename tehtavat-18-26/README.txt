Tehtävissä harjoiteltu REST-apin käyttöä:

- Reactin perusominaisuudet
- Fetchin käyttö
- REST-kutsut: GET-, DELETE-, POST- ja PUT- metodit
- formin käyttö
- conditional rendering
- dialogin käyttö

Sovelluksen käynnistys: npm run start
json-serverin käynnistys: json-server --watch db.json --delay 1000 --port 3004
